
# Authority Model

Example authority identifiers used in this repository are illustrative Project Alvorada test identifiers, not protocol-mandated universal roles.

- Papa Bear — role: Chief — authority: constitutional_authority
- Red-1 — role: Watcher — authority: coordinate_observe_analyze_recommend
- Blue-0 — role: unassigned_worker — authority: mission_scoped
- Mr. Gold — role: unassigned_worker — authority: mission_scoped
- Sienna-4 — role: unassigned_worker — authority: mission_scoped

Roles must not be self-assigned. Role changes require explicit authority-bearing records.
