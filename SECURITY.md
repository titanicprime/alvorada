
# Security Policy

Report implementation vulnerabilities and protocol defects through GitHub private vulnerability reporting for this repository.

## Scope

- **Implementation vulnerabilities** are flaws in code, automation, packaging, or repository operations.
- **Protocol defects** are flaws in schemas, dictionaries, codebooks, lineage rules, or validation logic.

Do not publish secrets, credentials, private prompts, confidential model data, or non-public evidence in issues, pull requests, or discussion threads.
